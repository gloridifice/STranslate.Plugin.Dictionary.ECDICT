# STranslate Anki Plugin

**[中文版本](README.md)**

> Disclaimer: This project was generated by DeepSeek v4 Pro, with minor review and testing by me. Feel free to submit an Issue if you find any bugs.

Save STranslate translation results as [Anki](https://apps.ankiweb.net/) cards with one click.

## Prerequisites

1. Install and run Anki
2. Install the [AnkiConnect](https://ankiweb.net/shared/info/2055492159) add-on in Anki (code: `2055492159`)
3. Restart Anki and visit `http://127.0.0.1:8765` to confirm AnkiConnect is running properly

## Installation

Open STranslate Settings → Plugins → Install .spkg

> Fallback: Extract the .spkg (essentially a .zip). Copy the plugin folder to the Plugins directory under the STranslate installation directory (where official plugins reside).

## Configuration

1. Add this plugin in STranslate
2. STranslate → Services → Add Anki and enable it
3. Select Anki in the services panel, click "Test Connection" to verify communication with Anki
4. Select the target deck and note type
5. Configure field mapping (source text → which field, translation → which field)
6. (Optional) Set tags and deduplication strategy

## Usage

1. Translate text in STranslate
2. Click the "Save" button next to the translation result
3. The text will be automatically added as an Anki card

![Add to Anki](images/add_to_anki.png)

## Deduplication

When "Allow Duplicates" is disabled, the plugin checks whether the target field already contains the same content before adding. If a duplicate is found, it will be skipped with a notification.

# Build

```shell
dotnet build Plugin\Plugin.csproj
```

The plugin output is generated under the `../.artifacts` directory.
